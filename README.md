# Product Management System

## About The Project
A Full stack product management app that communicates with a [database](https://github.com/AlteredStar/commerce-db), allowing users to manage products, customers, and transactions. Built with C#, ADO.NET, Npgsql, and PostgreSQL for the backend API and database. Utilizing React, Typescript, HTML, and CSS for the frontend app.

## Features
#### Add Products
Includes the functionality to automatically fetch product categories from the database and put them into a selectable dropdown list.

#### Search, Edit, & Delete Products
Able to search for products by name, then display the searched for product(s) in a table, and allows quick editing/deleting of rows through a few button presses.
